# Tripleten Around The U.S. Proyecto sprint

## Descripción

Around The U.S. es una aplicación web interactiva que permite a los usuarios ver y disfrutar una colección de tarjetas con imágenes de diferentes lugares. El usuario puede editar la información de su perfil, agregar nuevas tarjetas, dar "me gusta" a las publicaciones y visualizar las imágenes en una ventana emergente.

## Funcionalidad

- Edición del perfil del usuario.
- Visualización de tarjetas con imágenes y títulos.
- Agregar nuevas tarjetas mediante formularios emergentes.
- Apertura de imágenes en un popup.
- Validación de formularios.
- Botones de "Me gusta" en cada tarjeta.
- Cierre de ventanas emergentes mediante botón, superposición y tecla Esc.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- TypeScript
- Programación Orientada a Objetos (POO)
- Módulos ES6
- Manipulación del DOM
- Diseño responsivo
